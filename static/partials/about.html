ABOUT

LOOP was created to make Common Core data more actionable and to improve the feedback loop between teachers and students.

Studies have shown that specific, timely, descriptive feedback can greatly improve both teaching and learning effectiveness.

LOOP helps teachers use their standardized test data in useful ways and to track what students are doing well and what needs improvement, so that they can tell students HOW they can improve.
